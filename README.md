# Task & Project Management System

## 📌 Overview
Full‑stack web app for managing projects and tasks.  
Admins can create projects, assign tasks, and track progress.  
Members can view tasks on the Dashboard and mark them complete.

## 🚀 Features
- JWT authentication (Signup/Login)
- Role‑based access (Admin vs Member)
- Admin Panel for project creation and task assignment
- Dashboard with task cards and progress tracking
- Secure password hashing with bcrypt
- MongoDB database integration
- Deployed on Railway

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** Node.js, Express.js  
- **Database:** MongoDB  
- **Authentication:** JWT + bcrypt  
- **Deployment:** Railway
